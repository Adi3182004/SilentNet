# SilentNet Offline Messenger

A lightweight offline Android Studio app with:

- Splash screen
- Local login and register
- SHA-256 password hashing
- Offline chat UI
- Media attachments
- Local backup export
- Kotlin, Java, and Python integration through Chaquopy

## Open in Android Studio

Open the root folder `SilentNetOfflineMessenger` and wait for Gradle sync.

## Build notes

This project is pinned to:

- Android Gradle Plugin 8.5.2
- Gradle 8.7
- Kotlin 1.9.24
- Chaquopy 17.0.0
- minSdk 24
- JDK 17

The app is fully local and does not require a server.
